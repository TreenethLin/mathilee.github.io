project | PokeAPI Search

Dreaming of being a Pokemon trainer like Satoshi the protagonist of Pokemon is always my wish since I was young (This is the first Satoshi I know not the founder of Bitcoin Satoshi Nakamoto), I can tell you that I fall in love and call myself a big fan of this manga for a long time. So today I need to develop a simple web application that I can search for my favourite Pokemon such as Pikachu and Gyarados etc.


This project uses React JS and Tailwind CSS for frontend development and for the backend development I use Node JS to fetch data from PokeAPI such as the picture and weight of each Pokemon to show on my application
